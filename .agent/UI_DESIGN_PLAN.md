## Design System Overview

### Color Palette
- **Primary**: Blue (#0ea5e9) - Trust, Security
- **Secondary**: Indigo (#6366f1) - Authority
- **Success**: Green (#10b981) - Safe/Resolved
- **Warning**: Amber (#f59e0b) - Caution
- **Danger**: Red (#ef4444) - Emergency
- **Neutral**: Slate grays - Professional

### Typography
- **Headings**: Inter (Bold, 600-700)
- **Body**: Inter (Regular, 400-500)
- **Monospace**: JetBrains Mono (Code/IDs)

### Design Principles
1. **Clean & Minimal** - No clutter
2. **Data-Driven** - Clear metrics
3. **Action-Oriented** - Easy workflows
4. **Responsive** - Mobile-first
5. **Accessible** - WCAG 2.1 AA
